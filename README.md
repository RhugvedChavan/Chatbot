# Chatbot :- 

Developed an AI-powered conversational chatbot using LangChain and Mistral AI that enables users to interact with a large language model through a command-line interface. The chatbot maintains conversation history to provide context-aware responses, creating a more natural and interactive user experience.

# Objective :- 

1. Develop an intelligent chatbot capable of understanding and responding to user queries.
2. Integrate the Mistral Large Language Model using LangChain.
3. Maintain conversation memory for context-aware responses.
4. Demonstrate the practical application of Generative AI and Large Language Models.


# Key Features :- 

1. Interactive AI chatbot
2. Context-aware conversation using chat history
3. Real-time response generation
4. Environment variable support for secure API key management
5. Easy-to-use command-line interface


# Result :- 

1. Successfully developed a conversational AI chatbot capable of generating intelligent and context-aware responses.
2. The chatbot effectively remembers previous interactions during a session, improving conversation quality.
3. Demonstrated successful integration of LangChain with the Mistral AI language model for real-time text generation.

# Conclusion :- 

The project successfully demonstrates the implementation of a conversational AI chatbot using LangChain and Mistral AI. By maintaining conversation history and generating context-aware responses, the chatbot provides an engaging user experience. The project serves as a strong foundation for future enhancements such as a graphical user interface, voice interaction, document-based question answering, and integration with external APIs.


# Technology Used :-

1. Programming Language - Python
2. Framework - LangChain
3. Large Language Model - Mistral AI 
4. IDE - Visual Studio Code
